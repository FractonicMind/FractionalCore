## 🌟 What is Fractional Core?

Fractional Core (FC) revolutionizes information security by using mathematical diversity to create unforgeable proofs. Just as 1 can be expressed as `1`, `0!`, `7^0`, `cos(0)`, or `√1`, any data can be encoded through multiple equivalent mathematical expressions, creating a cryptographic fingerprint that's transparent yet secure.

### Core Principle
```javascript
// Traditional: One hash, one value
hash("data") = "5d41402..."

// Fractional Core: Multiple expressions, one truth
fc.encode(42) = ["42", "6×7", "√1764", "84/2", "2^5 + 10"]
// All equal 42, but the diversity proves authenticity
```

## 🚀 Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/[repository]/fractional-core.git
cd fractional-core

# Install dependencies
npm install

# Run tests
npm test

# Try examples
node examples/mathematical_captcha.js
```

### Basic Usage
```javascript
const FractionalCore = require('./fractional-core');
const fc = new FractionalCore();

// Encode value with mathematical diversity
const proof = fc.createProof(42);
// Returns: {
//   value: 42,
//   expressions: ['42', '6×7', '√1764', '84/2', '21×2'],
//   hash: 'a8f5f167...',
//   timestamp: 1234567890
// }

// Verify proof authenticity
const isValid = fc.verifyProof(proof); // true
```
